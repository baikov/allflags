Информация по API флагов и стран

API флагов стран:

  • Регион и субрегион
  • Код страны
  • Изображение флага (svg, png, webp, jpg)
  • Дата принятия флага
  • Пропорции
  • Цвета флага (название, hex, rgb, cmyk, Pantone)
  • Количество цветов
  • Элементы на флаге
  • Исторические флаги (изображение, годы, название)
  • Похожие флаги (те же группы цветов)
  • Флаги соседних стран

API стран:

  • Регион и субрегион
  • Код страны ISO 3166-1
  • Полное название (рус+англ)
  • Название страны (рус+англ) (+падежи: и, р, д, в, т, п)
  • Валюта (рус+англ) (название, коды iso, символ)
  • Кол-во населения (год, место в мире)
  • Интернет-домен
  • Телефонный код
  • Площадь территории (суши, воды), протяженность береговой линии
  • Соседние страны (+протяженность границы)
  • Язык (рус+англ)
  • Правительство и форма правления
  • Столица ((рус+англ))
